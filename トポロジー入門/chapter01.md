## (b)

$a=0, b=1, c=-1$ とする。
このとき、 $d(a, b) + d(a, c) = 2 < 4 = d(b, c)$ だから、
$\mathbb{R}$ の距離を定義しない。

## (c)

$a, b, c \in \mathbb{R}^n$ は

$$\begin{align*}
a_i &= 0 (1 \leq i \leq n)\\
b_i &= i (1 \leq i \leq n)\\
c_i &= - n - 1 + i (1 \leq i \leq n)\\
\end{align*}$$

を満たすとする。このとき

$$\begin{align*} 
d(a, b) + d(a, c) &= 2\\
d(b, c) &= n + 1
\end{align*}$$

$n > 2$ より $d(a, b) + d(a, c) < d(b, c)$ だから、 $\mathbb{R}^n$ の距離を定義しない。
